# Miniproject-2
HEB vs Walmart: Consumer Price Comparison

This project analyzes grocery pricing data to compare Walmart and H-E-B and determine which store offers lower prices for similar products. The analysis was performed in Python using a Jupyter Notebook and focuses on exploratory data analysis and basic machine learning.

What Has Been Done
	•	Imported multiple grocery datasets containing product names, categories, and unit prices.
	•	Used pandas to clean, inspect, and organize the data (checking shapes, missing values, and data types).
	•	Performed exploratory data analysis (EDA) to understand price distributions and product categories.
	•	Created visualizations using matplotlib and seaborn to compare pricing patterns.
	•	Built a supervised machine learning model to predict which store is cheaper for a given product.
	•	Used models such as K-Nearest Neighbors (KNN) and Random Forest with scikit-learn.
	•	Evaluated model performance using metrics like accuracy and mean squared error (MSE).

Tools and Libraries Used
	•	Python
	•	Jupyter Notebook
	•	pandas & numpy – data cleaning and manipulation
	•	matplotlib & seaborn – data visualization
	•	scikit-learn – machine learning models (KNN, Random Forest)

Opportunities for Improvement
	•	Improve data cleaning and matching of similar products between stores.
	•	Add feature engineering (price per unit, category encoding, brand comparison).
	•	Use classification models more consistently (e.g., Logistic Regression, Gradient Boosting).
	•	Perform cross-validation and hyperparameter tuning for better model accuracy.
	•	Create interactive dashboards (Plotly or Tableau) to visualize store price differences.
	•	Expand the dataset to include more stores or time-based pricing trends.

Overall, the project establishes a foundation for analyzing retail price competitiveness and consumer savings, with opportunities to expand into predictive analytics and pricing insights.
